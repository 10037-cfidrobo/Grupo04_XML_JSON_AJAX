XML (Extensible Markup Language)

XML es un lenguaje de marcado que se utiliza para almacenar y transportar datos de manera legible tanto para humanos como para máquinas. Está diseñado para ser autodescriptivo, lo que significa que los datos en un archivo XML están organizados en etiquetas que describen su contenido. Estas etiquetas se utilizan para definir elementos y estructuras de datos jerárquicas. XML se utiliza ampliamente para el intercambio de datos en aplicaciones web y servicios web.

AJAX (Asynchronous JavaScript and XML)

AJAX es una técnica de desarrollo web que permite enviar y recibir datos de un servidor en segundo plano, sin necesidad de recargar completamente la página web. Utiliza JavaScript para realizar solicitudes asíncronas al servidor y actualizar partes específicas de la página en función de los datos recibidos. Aunque su nombre incluye "XML", AJAX no está limitado a XML y puede trabajar con diferentes formatos de datos, como JSON.

JSON (JavaScript Object Notation)

JSON es un formato de intercambio de datos ligero y legible para humanos. Está basado en la sintaxis de objetos de JavaScript, por lo que es fácil de entender y utilizar en aplicaciones web. JSON se utiliza ampliamente para transmitir datos estructurados entre un servidor y un cliente. A diferencia de XML, JSON tiene una estructura más simple y es más eficiente en términos de tamaño y rendimiento. Puede representar datos en forma de pares clave-valor, matrices y objetos anidados.
